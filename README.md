The boilerplate files for creating a DevWatch blog.

```bash
src
│
└───drafts
│   │
│   └───subfolder1

│       │   file112.txt
│       │   ...
│
└───post
│   │   file021.txt
│   │   file022.txt
│   │
│   └───subfolder1
│       │   file111.txt
│       │   file112.txt
│       │   ...
│
└───sass
    │   file021.txt

```
